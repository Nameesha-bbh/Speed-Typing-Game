Instructions


  1. To start the game,you should choose the diffculty first and then click the button
  2. The default difficulty chosen is easy.
  3. The default timing for the speed typing game is 7 seconds.
  4. You can change the timings from 7 seconds to 5(medium) or 3(hard) seconds.
  5. Once you click the button the game will start immediately and the mouse will be focussed on the input box by default.
  6. Score is shown after every correct word is typed.
  7. Once you type the correct word , the next word and time automatically gets changed.
  8. Once the time is over, the game ends.
  9. Press the button to restart the game
